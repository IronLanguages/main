######################
  IronPython Hosting
######################

Provides information on IronPython and DLR Hosting APIs.  

The DLR hosting APIs provide a common way for hosting scripting languages
on .NET.  IronPython provides some specific extensions to these APIs to make
getting started and performing IronPython specific behaviors simpler.  This
section provides an overview of both the DLR specific APIs and the IronPython
extensions.  For deeply interacting with IronPython please see :_extending-index
